####### Projet QA Automation – Selenium & Python

## Description
Framework de tests automatisés en Python avec Selenium WebDriver,
ciblant le moteur de recherche Wikipédia.

Première approche. 

## Objectifs
- Démontrer des compétences en automatisation de tests QA
- Implémenter des tests H1 et de validation d'images
- Générer des rapports CSV exploitables dans Excel

## Technologies utilisées
- Python 3.x14
- Selenium WebDriver
- Git / GitHub

## Structure du projet
├── scripts/        # Scripts de test Selenium
├── données/        # Fichier de mots-clés
├── résultats/      # Rapports CSV générés
└── README.md

## Lancer le projet
```bash
python scripts/test_wikipedia.py
```
Les fichiers CSV sont générés dans le dossier `résultats/`.
